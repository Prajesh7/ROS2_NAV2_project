# ROS2_NAV2_project
Autonomous navigation in a custom map using ROS2 humble

# Dependencies:
    - ROS2 Humble
    - Ubuntu
    - numpy
    - opencv

# Launch Command:

    - ros2 launch my_slam_bringup my_slam.launch.py

# Note: Gazebo:
    - if you cant see anything in gazebo: go to "worlds" tab, select "scene" and un-check "shadows"
